Odoo "Islamic Datepicker" Module.

Installation Steps:- 1- Download Islamic Datepicker using this like Islamic Datepicker 2- Install it. Go to Setting>>local Module then install it.

Features

1- I will display Hijri Calender. 2- If you choose Hijri date, the module will reflect automatic Greogorain Calender and Vise versa... 3- It will be applied automatically on any exist or new object in Odoo. 4- You can see the Hijri Cander and Greogorain Calender in the same screen.
